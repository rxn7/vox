# VOX
C++ Voxel Engine (possibly a game in the future) made with modern OpenGL (core 4.3).

# Building
#### Linux is supported and Windows is sometimes tested (with Visual Studio 17 2022)

## Requirements
#### GLAD
- python3
- pip install jinja2

## Commands
### Linux
```bash
./scripts/configure.sh 
./scripts/debug.sh
```

### Windows
```bash
./scripts/configure.bat 
./scripts/debug.bat
```
